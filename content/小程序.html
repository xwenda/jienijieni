关于Taro热更新问题 ​

步骤 ​

  1. 安装

    js
    yarn add --dev @tarojs/plugin-vue-devtools

  2. 配置插件

    js
    // config/dev.js
module.exports = {
  ...
  // 增加这行
  plugins: ['@tarojs/plugin-vue-devtools']
}

  3. 编译

关于热重载失效或者只更新一次的问题 ​

js
// project.config.json
{
  ...
  "setting": {
    ...
    "compileHotReLoad": true, //这项设置为true
    ...
  },
  ...
}
